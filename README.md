Projeto de uma landing page institucional para a clínica odontológica OdontoRio, desenvolvida com foco em credibilidade, acolhimento e conversão de visitantes em pacientes.

O layout apresenta uma seção principal de destaque (Hero Section) com chamada de impacto, imagem humanizada e botão de ação para agendamento de consultas. A navegação foi projetada para oferecer acesso rápido às principais informações da clínica, incluindo apresentação institucional, serviços oferecidos e canais de contato.

O projeto prioriza boas práticas de desenvolvimento Front-End, utilizando HTML5 e CSS3 para construção de uma interface responsiva, moderna e acessível. A identidade visual emprega tons de azul e verde associados à saúde, confiança e bem-estar, reforçando o posicionamento da marca.

Entre os principais objetivos do projeto estão:

Apresentar a clínica de forma profissional e acolhedora;
Destacar a qualidade dos serviços odontológicos;
Facilitar o contato e o agendamento de consultas;
Proporcionar excelente experiência em dispositivos desktop, tablet e mobile;
Aplicar conceitos fundamentais de estruturação semântica, layout responsivo e organização de código.

Este projeto foi desenvolvido como exercício prático de Front-End, aplicando conceitos de design responsivo, hierarquia visual, posicionamento de elementos, componentes reutilizáveis e construção de interfaces modernas para a web.
